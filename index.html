Inkscape Symbols Libraries

5,600+ ready-to-use symbols for Inkscape

View on GitHub Download ZIP

What is this?

A curated collection of 48 symbol libraries containing over 5,600 symbols for use with Inkscape, the free and open-source vector graphics editor.

Quick Install

Copy the SVG files to your Inkscape symbols directory:

Linux

git clone https://github.com/wpgaurav/Inkscape-Symbols-Libraries.git ~/.config/inkscape/symbols/

macOS

git clone https://github.com/wpgaurav/Inkscape-Symbols-Libraries.git ~/Library/Application\ Support/org.inkscape.Inkscape/config/inkscape/symbols/

Windows

Download the ZIP file and extract the SVG files to %APPDATA%\inkscape\symbols\
